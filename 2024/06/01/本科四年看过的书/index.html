本科四年看过的书

Posted on 2024-06-01 Edited on 2024-06-03 In 回忆
本科四年看过的一些书, 不一定完全看完了

快毕业了, 记录一下还有印象的一些书.

计算机

C++大学教程

C++课的参考书, 砖头一样的书, 个人感觉没有任何亮点

C++ primer

网上推荐的书, 我也主要参考这本, 个人感觉中规中矩, 也没有很大的亮点. 后来感觉初学不能太深究语言的细节, 没有其他背景知识根本不知道为什么要这么设计.

Compilers

编译技术的参考书, 封面是一只龙, 称为龙书. 英文究极长难句, 中文译的相当烂, 读起来太痛苦了.

Computer Networking A Top-Down Approach

计算机网络课的参考书

Modern Operating System

操作系统课的参考书, 但是认真读了原版几个章节, 得出的结论就是: 以后都不想读这种百科全书式的书了.

Database System Concepts

数据库的参考书, 里面好多内容没搞懂

算法导论

非常有名的书, 收了一本中文版收藏用, 一点没读

UNIX环境高级编程

传说中的APUE, 适合拿来查阅, 不适合阅读.

Linux-Unix 系统编程手册

APUE的平替, 网上风评不错, 作者是man的维护者, 读了上册. 这些晦涩庞大的内容很难讲的出彩, 适合拿来查阅.

TCP IP 网络编程

好书, 讲的比较通俗. 最好需要学过计算机网络, 懂一点unix的api. 示例代码非常多, 前置知识少.
把书里的代码抄了一份

深入理解计算机系统

好书. 相比于其他比较出名的书(比如现代操作系统, APUE), 可读性非常高. 内容编排合理, 行文流畅, 中文版翻译的很好, 还有许多lab, 网上也有众多环境配置以及解答的文章. 相当推荐.

统计学习方法

大家都推荐的书, 补了前面几章.

机器学习

封面画了个西瓜的那本. 大家都推荐的书, 补了前面几章. 没上过机器学习课, 补充一些知识.

数据挖掘导论

数据挖掘的参考书

算法设计与分析基础

算法课的参考书, 基本没翻过?

计算机网络 谢希仁

考研计算机网络的参考书, 没仔细看

鸟哥的Linux私房菜

百科全书, 个人感觉一般, 不如上google搜索

STL源码剖析

网上推荐挺多的书, 实际读下来收获并没有想象中的大. 学到的东西有

  • allocator
  • 各种 typedef, 比如difference_type, pointer, reference, size_type等等, 之前在pmp, openmesh读到这些内容感觉莫名其妙, 原来是为了能够使用stl算法
  • deque: 之前一直误解了, 读完才知道deque的iterator是random access iterator, 以及一直以为它和std::list没有什么区别.

Polygon Mesh Processing

几何处理方向的参考书, 暂时没有中文版, 读了前面几章基础知识, 还有smoothing, parameterization, deformation相关的章节.

当时还写了一个半边数据结构zmesh, 借鉴的pmp, 相比之下, zmesh的特点有:

  • 实现了smart handle, pmp的东西都写在一个文件里, 没法实现smart handle. smart handle配合range-based for写起来是真的爽!
  • 用property system统一表示各种属性, 包括网格的拓扑连接关系和几何信息, 当然也可以自定义property.
  • 用traits和constexpr if实现各类circulator, 减少重复的代码, 提高扩展性. pmp是每种circulator都写成一个类.

Fundamentals of Computer Graphics

图形学虎书, 第一本认真阅读的课外的英文书.

数学

数学分析教程 史济怀

科大数院考研参考书, b站有史济怀老师的网课, 配套使用, 效果很好. 个人感觉很好的数分课本. 不过刘思齐老师对这本书评价较低, 因为有错别内容. 另: 刘思齐老师的课让我见识到我和顶尖学校的学生差距实在太大了.

数学分析新讲 张筑生

虽然名字是新讲, 但已经是好久前的书了. 最喜欢的数学分析教材, 级数部分讲的特别好. 场论那里引入了微分流形之类的, 反正我读不懂. 除此之外, 其他地方讲的很好.

数学分析习题课讲义 谢惠民

盲猜数院考研人手一套? 做了上册, 当时没带手机去图书馆, 没得查答案, 经常几道题一做就是一整个早上, 想不出来硬是想出来, 很有成就感. 题目质量相比裴礼文高了很多.

数学分析中的典型问题与方法 裴礼文

俗称裴砖🧱, 盲猜数院考研人手一本. 大概做了一半. 有的题目其实是工科考研数学的难度, 不太有分析的味道.

高等代数 丘维声

俗称丘砖🧱, 上册完全就是工科线性代数的内容. 读完的感悟是: 数院一直在创新数分和高代的讲授方式, 出现了许多适合自学的教材, 而理工科一直在用那几本很烂的教材, 建议工科生用丘砖上当作自学材料.

高等代数白皮书

原来的名叫啥忘了, 总之就是复旦的那本, 刷完jordan标准形和二次型之后就放弃了, 前面几章习题难度适中, 同样适用于工科的考研数学. b站有谢启鸿老师的网课, 全程板书. 谢老师自称是三流的数学家, 但希望自己是一流的数学教师. 泪目😭

线性代数 - 李尚志

科大数院考研参考书. 网上评价两极分化, 因为这本书行文非常流畅, 适合自学, 同时勘误很多. 个人读过最好的线性代数教材(可读性最高, 适合自学), 不过需要搭配网上的勘误. 基本把课后习题全刷了, 毕竟是官方指定参考书.

抽象代数 - 丘维声

紫色封面的一小本, 可以在Galois理论那里看到丘老师特有的自己引用自己.

近世代数引论 - 冯克勤, 李尚志, 章璞

科大数院复试参考书. 个人觉得可读性为0, 需要有老师带着学, 好在b站有章璞老师的网课. 有一本搭配的习题, 近世代数300题.

A First Course in Abstract Algebra

中文的抽象代数都是小薄本, 内容相当精炼. 国外的都是大部头, 这本是知乎推荐的比较多的一本, 当时在图书馆六楼自学, 刚好看到有这本书, 就借回来读了, 不过只读到了环论.

这本书特点是例子多, 语言简明(我都能看懂). 当然, 内容一多, 就难以抓住重点, 这点见仁见智.

stein 分析系列

只读了傅里叶分析和复分析. 这两本也是评价最高的两本. 非常适合自学. 有数学分析的基础就可以读下去. 总之, 神中神!

傅里叶分析后面有一张讲了fft, 之后又找了资料, 用fft实现了多项式乘法(之前一直没搞懂这个多项式乘法), 觉得这里面的原理非常酷🤔. 与kt分享之后, 他表示从z变换的角度看这是非常显然的事🫡!
